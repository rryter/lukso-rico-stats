Level 1 - Proxy Account

ERC 725 is a proposed standard for blockchain-based identity authored by Fabian Vogelsteller, creator of ERC 20 and Web3.js. ERC 725 describes proxy smart contracts that can be controlled by multiple keys and other smart contracts.


Level 2 - Key Manager

Right now all you have is a plain and simple ERC275 proxy account. The ERC734 key manager standard allows for more complex management of an ERC725 proxy account.
